# TermiWeb Plan

Bootstrap v1 of the shared browser terminal: docs, repo workflow, auth gate, instance/session management, terminal backend, and a usable all-device browser shell.

## Milestones

- [x] M1: Repository bootstrap and docs baseline
- [x] M2: Shared terminal backend and session model
- [x] M3: Browser UI with instances, auth, and mobile controls
- [x] M4: Local verification and launch polish

Run real multi-device/manual UX checks against the live browser terminal, then tighten shared-instance behavior, selection mode, and mobile controls based on what feels bad in practice.
