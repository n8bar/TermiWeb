# TermiWeb Plan

## Current Focus

Bootstrap v1 of the shared browser terminal: docs, repo workflow, auth gate, tab/session management, terminal backend, and a usable desktop/mobile browser shell.

## Milestones

- [x] M1: Repository bootstrap and docs baseline
- [x] M2: Shared terminal backend and session model
- [x] M3: Browser UI with tabs, auth, and mobile controls
- [x] M4: Local verification and launch polish

## Next Action

Run real multi-device/manual UX checks against the live browser terminal, then tighten session ergonomics and mobile controls based on what feels bad in practice.
